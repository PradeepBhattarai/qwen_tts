Streaming Voice Clone Test Console

Drive the Qwen3 websocket stream directly from the browser.

Upload a reference clip or point at a server-local file, send one websocket request, and the page will schedule raw PCM chunks as they arrive. This UI is aimed at the `wss://.../api/v1/tts/stream` contract exposed by the FastAPI service.

Stream Request

Binary websocket frames are expected to be raw mono `pcm_s16le` chunks. The page converts them to `AudioBufferSourceNode` segments and schedules them in sequence for low-friction manual testing.
